# Vgit

## Simple git remote url manager

#### Installation

* You can move vgit program to /bin/ directory. You'll need run:

```bash
cp vgit.py vgit
chmod a+x vgit
mv vgit /usr/bin/
```

### Run vgit

* Go to a git repository and run following command:

```bash
vgit
```

and it will be automatically detected as a Git repository
